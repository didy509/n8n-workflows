# AI Email Classifier

## À propos

Workflow réalisé dans le cadre d'une formation pratique sur n8n (Udemy) afin de développer mes compétences en automatisation et en IA générative.

## Description

Ce workflow automatise la classification d'e-mails grâce à l'API OpenAI.

## Technologies

- n8n
- OpenAI API
- Gmail

## Objectif

Mettre en pratique les automatisations avec les LLM et les intégrations d'API.

> Ce projet a été réalisé dans le cadre de ma montée en compétences en IA générative et automatisation.
