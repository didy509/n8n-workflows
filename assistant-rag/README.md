# RAG Assistant

## À propos

Workflow réalisé dans le cadre d'une formation pratique sur n8n (Udemy) afin de développer mes compétences en IA générative, RAG et automatisation.

## Description

Ce workflow met en œuvre un assistant basé sur le Retrieval-Augmented Generation (RAG) pour enrichir les réponses d'un LLM avec une base de connaissances.

## Technologies

- n8n
- OpenAI API
- RAG
- LLM

## Objectif

Mettre en pratique les concepts de RAG et développer mes compétences en IA générative et automatisation.
